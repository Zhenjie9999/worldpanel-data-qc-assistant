# Worldpanel FMCG QC Rules

Use these rules as review hypotheses. Validate the measure definition, base, period, and unit before escalating a finding.

## Core metric checks

- **Share:** Components normally sum to about 100% within a declared universe. Allow stated rounding and "Other"; inspect missing brands, duplicated segments, inconsistent universe, or percentage/decimal formatting.
- **Penetration:** Must be between 0% and 100%. Compare buyer base and population definition before comparing across sources.
- **Price / average price:** Check currency, pack size, unit (per kg/L/100g/item), promotion treatment, and weighted versus simple average. Flag only values materially far from their product-period peer group or impossible unit conversions.
- **Volume / spend / value:** Check non-negative values, consistent units, and expected identities such as value approximately equal to volume times price when the definitions match.
- **Buyers and volume per buyer:** Where definitions align, volume should approximately equal buyers multiplied by volume per buyer. Verify panel projection and reporting units first.
- **Growth:** Confirm comparison period, MAT/YTD terminology, denominator, and sign. A percentage point change is not the same as percentage growth.

## Deliverable checks

- Match PPT/PDF labels, headlines, footnotes, category names, brands, time periods, and numeric values to the source workbook.
- Flag an apparent mismatch only when the source candidate is unique or the candidate alternatives create a meaningful conflict. Surface low-confidence matches for manual review rather than claiming an error.
- Check charts for labels inconsistent with displayed values, axis units inconsistent with captions, incomplete totals, and conclusions contradicted by data.
- Check that all pages/sheets requested in scope were reviewed; clearly record excluded or unreadable content.

## Severity guide

- **Critical:** Material incorrect reported number, wrong period/category, or source conflict that can change the client conclusion.
- **High:** Strong logic violation, implausible outlier, missing/incorrect key annotation, or unresolved low-confidence source match on a key metric.
- **Medium:** Inconsistent formatting, rounding beyond tolerance, secondary label issue, or incomplete evidence requiring analyst review.
- **Low:** Cosmetic inconsistency or a weak signal without confirmed material impact.
